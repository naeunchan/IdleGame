# AGENTS.md

## Project

Original 2D pixel-art idle/tycoon game about dog developers running a cozy software company.

## Style rules

- Cozy, warm, readable, original pixel art
- Inspired by classic indie life-sim comfort, but never copy assets or compositions
- Mobile-first readability
- Favor simple silhouettes and limited palettes
- Optimize for solo-dev production speed

## Art rules

- Use original characters, props, UI, and tile layouts
- Avoid direct resemblance to any specific commercial game asset
- Keep sprites modular
- Prefer 16x16 / 24x24 / 32x32 grids depending on object class
- Keep animation frame counts small and reusable

## Output rules

- Always provide a short plan first for broad tasks
- Prefer implementation-ready specs over abstract discussion
- When proposing assets, mark priority as must / should / later
- When proposing art, include technical constraints for HTML5/web
- All outputs must be in Korean unless explicitly requested otherwise
